Tarkoituslause

Tarkoituslause (finaalilause) kertoo tarkoituksen, miksi jotain tehdään.

Tarkoituslause rakentuu: (για) + να + konjunktiivi.

για + να + konjunktiivin kertamuoto/kestomuoto

Jos tarkoitus-rakenteessa on verbi mukana, käytetään να + konjunktiivia.

για-prepositio ei ole pakollinen.

Konjunktiivi voi olla kesto- tai kertakonjunktiivi. 

Πήγε για να αγοράσει.  "Hän meni ostaakseen."
= Πήγε να αγοράσει.
Να έρχεσαι πιο συχνά να σε βλέπουμε.  "Käy useammin, että nähtäisiin."
Τι σκοπεύει να κάνει; "Mitä häin aikoo tehdä?"

Kielletään μη(ν)-kieltosanalla. 

Διαβάζω τις λέξεις πολλές φορές για να μη τις ξεχνάω. "Luen sanat moneen kertaan, etten unohtaisi niitä."
Αν μου πει ο καθηθητής μου να μη γελάσω στο μάθημα δε θα γελάσω. "Jos opettajani käskee minua, etten nauraisi tunnilla, en naura tunnilla."
Σκέφτεστε να μη τρώτε τίποτα;  "Ajatteletteko olla syömättä mitään?"
Έφυγε για να μη με βοηθήσει. "Hän lähti, ettei auttaisi minua."
Έφυγε για να μη δουλέψει στον κήπο.  "Hän lähti ettei tekisi töitä puutarhassa."
Δεν αγοράζω τσιγάρα για να μη καπνίζω.  "En osta tupakkaa, jotta en polttaisi."
Έφυγα για να μην τον συναντήσω. "Hän lähti, ettei tapaisi tätä."
Τρέχει για να μην αργήσει. "Hän juoksee, ettei myöhästyisi."

για + substantiivi tai pronomini akkusatiivissa 

Πήγα στη θάλασσα για μπάνιο.  "Menin merella aurinkoa ottamaan."
Ήρθα εδώ για σένα. "Tulin tänne sinun takiasi."
