Infinitiivi

Nykykreikassa ei ole erillistä muotoa infinitiiville. Sen sijaan käytetään partikkelia να + konjunktiivin kertamuoto tai kestomuoto.

Wikipediassa käytetään nimitystä dependentti, koska verbimuoto edellyttää, että sitä edeltää partikkeli θα tai να, tai apuverbi έχω.

Sanakirjoissa infinitiivin sijasta käytetään indikatiivin preesensin 1. persoonaa, esim. γράφω "kirjoitan"

Nämä kaikki tarkoittavat "kirjoittaa": 

να γράφω
να γράφεις
να γράφει
να γράφουμε
να γράφετε
να γράφουν
 
να γράψω
να γράψεις
να γράψει
να γράψουμε
να γράψετε
να γράψουν
 
Θέλω να γράφω. "Haluan kirjoittaa (yleensä)."
Θέλεις να γράφεις.  "Haluat kirjoittaa (yleensä)"
jne.
Θέλω να το γράψω. "Haluan kirjoittaa sen"
Θέλεις να το γράψεις.  "Haluat kirjoittaa sen"
jne.

Verbin tekijä voi olla eri kuin kohde. 

Θέλω να γράφεις. "Haluan sinun kirjoittavan."
Θέλεις να γράφω. "Haluat minun kirjoittavan."
Της αρέσει να δουλεύει. "Hänestä on mukava tehdä työtä."
Σας προσκάλεσα να βγούμε έξω ένα βράδυ.  "Kutsuin teidät syömään kanssamme ulkona joku ilta."
Αποφάσισα να μην πηγαίνω σε καλά εστιατόρια.  "Päätin, etten mene hyviin ravintoloihin."
Δεν χρειάζεσαι να αδυνατίσεις.  "Ei sinun tarvitse laihtua."
Ξέχασα να σου πω πως θα 'ναι μαζί μας και μια φίλη μου.  "Unohdin sanoa sinulle, että eräs ystävättäreni tulee kanssamme."
